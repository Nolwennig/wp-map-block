Editor block for easily inserting an OpenStreetMap embedded map into your post content.

== Description ==

Map Block is a WordPress plugin that registers a new [editor block](https://developer.wordpress.org/plugins/blocks/) for easily inserting an [OpenStreetMap](https://www.openstreetmap.org/) embedded map into your post content.

== Installation ==

Install Map Block via the plugin directory, or by uploading the files manually to your site's `wp-content/plugins` directory.
